# XSchemOptimizer

Usage:
```
pip install xschemoptimizer
# refer to examples/
```

## Commands

### Build the library:
```bash
cargo build --release
```

### Run tests:
```bash
cargo test
```

### Run benchmarks:
```bash
cargo run --bin benchmark
```

### Build Python bindings:
```bash
maturin develop
```

## Usage

See examples in `examples/` directory.
