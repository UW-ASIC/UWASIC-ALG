# XSchemOptimizer

Circuit parameter optimization library for XSchem analog design workflows.

## Commands

### Build the library:
```bash
cargo build --release
```

### Run tests:
```bash
cargo test
```

### Run benchmarks:
```bash
cargo run --bin benchmark
```

### Build Python bindings:
```bash
maturin develop
```
